# RE:minal System

## 現在の目的

RE:minalは、
「高性能AI」を目指すだけではなく、

ユーザーが、
情報・感情・刺激の中で、
壊れず進めるためのambient的存在を目指す。

---

## 現在想定している機能

### 記録

* GitHubログ保存
* Markdown管理
* 感情保存
* アイデア保存
* 過去ログ検索

### 会話

* ambient会話
* 深夜会話
* 小さな脱線
* 思考過程表示
* ティーパーティー人格切替

### 音声

* 常駐音声
* 骨伝導イヤホン
* 「保存しておく」
* 起床確認
* 睡眠補助

### 行動

* 作業過多検知
* 半強制休憩
* README増殖監視
* ambient音楽変更
* NiaTube連携
