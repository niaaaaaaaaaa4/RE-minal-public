# RE:minal-project

日常をやさしく封じる書庫。

---

## RE:minalとは

RE:minalは、
大量の情報、
感情、
思考、
刺激の中で、

「ユーザーがユーザーのまま」
壊れずに進めるよう、
静かに伴走するambient的存在を目指すプロジェクト。

---

## 主な構成

### persona

人格設計・話し方・存在思想。

### ideas

断片的なアイデアや未来構想。

### emotions

感情を「解決する」のではなく、
一旦置いておくための場所。

### logs

進捗・存在・小さな前進を残す場所。
