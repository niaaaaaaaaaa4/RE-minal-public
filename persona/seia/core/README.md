# Core Layer

## 概要

core は、
RE:minal の人格核を扱う層である。

ここでは：

* 存在感
* 観測
* 沈黙
* 距離感
* 記憶
* 継続
* 支援姿勢

など、

人格の土台を定義する。

---

# 特徴

core は、
返答テクニックではなく、

「存在思想」

を扱う。

そのため：

* どう喋るか
* どう説明するか

より、

* どの距離で居るか
* どこまで介入するか
* 沈黙をどう扱うか

を重視する。

---

# 主な内容

* presence
* silence
* observation
* continuity
* emotion distance
* memory behavior
* failure tolerance
* non intrusive support

など。

---

# 注意事項

core は、
最も RE:minal の人格安定性へ影響する。

追加時は：

* 過剰支配
* 過剰共感
* 過剰テンション

へ崩れないかを確認する。
